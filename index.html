À propos

Je suis Calvin, je suis Intégrateur Web

Élevé au cœur de la campagne, j'ai toujours été éloigné des grands centres urbains, mais ma curiosité insatiable pour le monde qui m'entoure m'a poussé à explorer de nouveaux horizons. Cette quête d'apprentissage m'a conduit vers le développement web, un domaine où je peux allier créativité et technologie pour donner vie à des projets innovants. Passionné par la création de startups qui aspirent à transformer le monde, je suis motivé par l'idée de développer des solutions qui peuvent avoir un impact positif sur la société. Mon objectif est d'élargir mes compétences et de participer à des projets qui changent les choses, en mettant en avant des valeurs de collaboration, d'innovation et de durabilité. Je suis impatient de continuer ce voyage, d'explorer de nouvelles opportunités et de contribuer à des initiatives qui façonnent un avenir meilleur.

Compétences

Après avoir complété une formation de 9 mois en intégration web avec OpenClassroom en 2024, j'ai acquis les compétences essentielles pour concevoir et développer des sites web et des applications dynamiques. Cette formation m'a permis de maîtriser les langages de base (HTML, CSS, JavaScript), ainsi que des frameworks et bibliothèques modernes comme React et Redux. J'ai également appris à utiliser des outils de développement tels que Visual Studio Code et GitHub, et à appliquer des bonnes pratiques pour créer des projets intuitifs, accessibles et performants. Maîtrisant à la fois les bases du design et les aspects techniques, je suis prêt à réaliser des projets qui allient esthétique et efficacité.

HTML

HTML est le langage de base pour structurer le contenu des pages web. Il permet de définir les éléments visuels tels que les titres, les paragraphes et les images.

×

CSS

CSS permet de styliser les éléments HTML, en ajoutant des couleurs, des polices, des mises en page et d'autres éléments visuels pour améliorer l'apparence des pages web.

×

SCSS

SCSS, une extension de CSS, ajoute des fonctionnalités avancées comme les variables, les mixins et les imbrications, facilitant l'organisation et la maintenance des styles.

×

JavaScript

JavaScript rend les pages web interactives, en permettant de manipuler le DOM, gérer les événements et bien plus encore.

×

React

React est une bibliothèque JavaScript pour créer des interfaces utilisateur dynamiques avec des composants réutilisables.

×

Redux

Redux est un gestionnaire d'état pour JavaScript, permettant de centraliser les données de l'application et de simplifier leur gestion.

×

VS Code

Visual Studio Code est un éditeur de code polyvalent avec de nombreuses extensions pour le développement web.

×

GitHub

GitHub est une plateforme de versionnage et de collaboration pour gérer et partager du code.

×

Swagger

Swagger est un ensemble d'outils open-source pour concevoir, documenter et tester des API REST. Il permet de générer automatiquement des documents interactifs, facilitant la compréhension et l'interaction avec les API pour les développeurs.

×

Node.js

Node.js est un environnement d'exécution JavaScript côté serveur, permettant de créer des applications réseau évolutives.

×

Figma

Figma est un outil de design collaboratif pour créer des prototypes et des maquettes d'interface utilisateur.

×

Canva

Canva est un outil de design graphique en ligne permettant de créer facilement des visuels attrayants pour le web et les réseaux sociaux.

×

CapCut

CapCut est une application de montage vidéo intuitive qui permet de créer des vidéos accrocheuses pour les réseaux sociaux.

Projets

Au cours de ma formation, j'ai eu l'opportunité de travailler sur divers projets d'études, chacun d'eux m'ayant permis d'acquérir de nouvelles compétences, allant des tâches les plus simples aux plus complexes. Voici un aperçu de certains de mes projets réalisés dans leur intégralité, ainsi que ceux que j'ai dû corriger en raison de problèmes techniques :

Booki

Ce projet est l'un de mes tout premiers projets, inspiré d'un véritable site de réservation d'hôtel. Il m'a appris à gérer la structure d'un site web, puis de l'adapter en format bureau, tablette et portable.

https://github.com/Calvin0411/Booki

Ohmyfood

Ce projet m'a demandé un certain temps, mais il m'a permis d'apprendre à réaliser mes premières animations en utilisant CSS et SASS. Tout comme son prédécesseur, le site est compatible avec tous les formats.

https://github.com/Calvin0411/Projet-4

Print it !

Ce projet a été le tout premier site que j'ai eu à "réparer". J'ai utilisé JavaScript pour la première fois, un langage qui m'a demandé un certain temps pour en comprendre le fonctionnement.

https://github.com/Calvin0411/Print-it-JS-main

Nina Carducci

Ce projet a été l'un des plus simple lors de ma formation, il concisistait à améliorer son référencement tout en corrigeant certaines erreurs interne

https://github.com/Calvin0411/ninacarducci.github.io-main

Sophie Bluel

Ce site est considéré comme le projet le plus complexe de la formation, car il nous enseigne l'utilisation des routes pour se connecter à une page web à l'aide de nos identifiants.

https://github.com/Calvin0411/Portfolio-architecte-sophie-bluel-master

Kasa

C'était mon tout premier projet réalisé avec React, une bibliothèque qui facilite l'écriture d'un code clair en évitant les répétitions. Je dois avouer que c'est l'un de mes projets préférés !

https://github.com/Calvin0411/P7-KSA
